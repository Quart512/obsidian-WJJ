## 🎮 코어 시스템

### 1. 이벤트 시스템

#### 이벤트 디스패처
```cpp
// 글로벌 이벤트 매니저
UCLASS()
class UEventManager : public UObject
{
    GENERATED_BODY()
    
    // 이벤트 델리게이트
    DECLARE_MULTICAST_DELEGATE_TwoParams(FOnDamageTaken, AActor*, float);
    DECLARE_MULTICAST_DELEGATE_OneParam(FOnActorDeath, AActor*);
    
    static FOnDamageTaken OnDamageTaken;
    static FOnActorDeath OnActorDeath;
    
    // 이벤트 발생
    static void BroadcastDamageTaken(AActor* Victim, float Damage);
    static void BroadcastActorDeath(AActor* DeadActor);
};
```

#### 사용 예시
```cpp
// 구독
UEventManager::OnActorDeath.AddDynamic(this, &AMyGameMode::OnActorDeath);

// 발생
UEventManager::BroadcastActorDeath(DeadEnemy);
```

### 3. 퀘스트 시스템

USTRUCT(BlueprintType)
struct FQuestObjective
{
    GENERATED_BODY()
    
    UPROPERTY()
    FString Description;
    
    UPROPERTY()
    int32 CurrentProgress;
    
    UPROPERTY()
    int32 RequiredProgress;
    
    bool IsComplete() const { return CurrentProgress >= RequiredProgress; }
};

USTRUCT(BlueprintType)
struct FQuest
{
    GENERATED_BODY()
    
    UPROPERTY()
    FString QuestName;
    
    UPROPERTY()

    UPROPERTY()
    bool IsCompleted;
};

## 🤖 AI 시스템

### 1. 행동 트리 구조

#### 블랙보드 키
```cpp
// 블랙보드 키
- TargetActor (Object)
- TargetLocation (Vector)
- State (Enum)
- IsInCombat (Bool)
- PatrolIndex (Int)
```

#### 행동 트리
```
Root
├── Selector (State Machine)
│   ├── Sequence (Idle)
│   │   ├── Wait
│   │   └── SetState (Patrol)
│   ├── Sequence (Patrol)
│   │   ├── MoveTo (Patrol Point)
│   │   └── SetState (Chase)
│   ├── Sequence (Chase)
│   │   ├── MoveTo (Player)
│   │   └── SetState (Attack)
│   └── Sequence (Attack)
│       ├── Attack
│       └── SetState (Idle)
```
