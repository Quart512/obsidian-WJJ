# Advanced Tables 플러그인 가이드

> 마크다운 테이블 쉽게 편집하기

**카테고리**: 유틸리티 & UI 개선
**난이도**: 낮음
**추천도**: ⭐⭐⭐

---

## 📋 개요

### **Advanced Tables란?**
마크다운 테이블 생성과 편집을 쉽게 해주는 플러그인

### **주요 기능**
- ✅ 자동 줄 맞춤
- ✅ 탭 키 네비게이션
- ✅ 열 추가/삭제
- ✅ 정렬
- ✅ 포맷팅

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Advanced Tables"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Advanced Tables
- Automatic table formatting: ON
- Use tab to navigate: ON
```

---

## 🎨 기본 사용법

### **1. 테이블 생성**
```markdown
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| 내용1 | 내용2 | 내용3 |
```

### **2. 탭으로 이동**
```
Tab: 다음 셀로 이동
Shift+Tab: 이전 셀로 이동
Enter: 새 행 생성
```

### **3. 자동 정렬**
```
입력 후 자동으로 줄 맞춤
```

---

## ⚡ 고급 기능

### **1. 정렬**
```
Command Palette → Advanced Tables: Sort table
```

### **2. 열 추가/삭제**
```
Command Palette → Advanced Tables: Add column
Command Palette → Advanced Tables: Remove column
```

### **3. 행 추가/삭제**
```
Command Palette → Advanced Tables: Add row
Command Palette → Advanced Tables: Remove row
```
