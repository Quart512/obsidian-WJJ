# Make.md 플러그인 가이드

> 직관적인 편집 & 정리 기능

**카테고리**: 유틸리티 & UI 개선
**난이도**: 중간
**추천도**: ⭐⭐⭐

---

## 📋 개요

### **Make.md란?**
직관적인 편집 기능, Spaces 개념을 통한 노트 그룹화, Contexts 기능을 통한 노트 데이터베이스화 등 다양한 UI 및 정리 기능을 제공하는 플러그인

### **주요 기능**
- ✅ Spaces (노트 그룹화)
- ✅ Contexts (노트 데이터베이스화)
- ✅ 직관적 편집
- ✅ UI 개선
- ✅ Keep the Rhythm

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Make.md"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Make.md
- Enable Spaces: ON
- Enable Contexts: ON
```

---

## 🎨 주요 기능

### **1. Spaces**
```
노트를 그룹화하여 관리
- 프로젝트별 Spaces
- 주제별 Spaces
```

### **2. Contexts**
```
노트에 메타데이터 추가
- 태그
- 날짜
- 진행 상태
```

### **3. Keep the Rhythm**
```
작업 흐름 유지
- 마지막 작업 위치 기억
- 빠른 전환
```
