# Calendar 플러그인 가이드

> 일일 노트를 위한 월별 탐색기

**카테고리**: 노트 탐색 & 정리
**난이도**: 낮음
**추천도**: ⭐⭐⭐⭐

---

## 📋 개요

### **Calendar란?**
사이드바에 월별 캘린더를 추가하여 일일 노트를 쉽게 탐색하고 생성하는 플러그인

### **주요 기능**
- ✅ 월별 캘린더 뷰
- ✅ 일일 노트 빠른 생성
- ✅ Periodic Notes 연동
- ✅ 주간/월간 노트 지원
- ✅ 시각적 진행 추적

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Calendar"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Calendar
- Start week on: Monday (또는 Sunday)
- Show week number: ON/OFF
- Create daily note on click: ON
```

---

## 🎨 기본 사용법

### **1. 캘린더 열기**
```
리본 아이콘 클릭 → 캘린더 패널
```

### **2. 일일 노트 생성/열기**
```
날짜 클릭 → 노트 자동 생성
이미 존재하면 → 노트 열기
```

### **3. 주간/월간 노트**
```
주 번호 클릭 → 주간 노트
월 이름 클릭 → 월간 노트
```

---

## ⚡ Periodic Notes 연동

### **설정**
```
Settings → Calendar
- Weekly note: ON
- Monthly note: ON
```

### **템플릿 설정**
```
Settings → Periodic Notes
- Daily: Templates/Daily Template.md
- Weekly: Templates/Weekly Template.md
- Monthly: Templates/Monthly Template.md
```

## 🎯 WJJ 활용 예시

### **1. 데일리 루틴 추적**
```markdown
# Calendar로 데일리 노트 관리

1. 캘린더에서 오늘 날짜 클릭
2. 자동으로 템플릿 적용
3. 습관 체크리스트 완료
4. 회고 작성
```

### **2. 위클리 리뷰**
```markdown
# 주간 노트 자동 생성

1. 주 번호 클릭
2. Weekly Template 자동 적용
3. 주간 목표 작성
4. 주간 회고 작성
```

---

## 💡 시각적 추적

### **1. 완료된 날짜 표시**
```markdown
Settings → Calendar
- Show dots for days with notes: ON
```

### **2. 특정 태그 표시**
```markdown
# 데일리 노트에 태그 추가
#daily #complete

→ Calendar에 점으로 표시
```
