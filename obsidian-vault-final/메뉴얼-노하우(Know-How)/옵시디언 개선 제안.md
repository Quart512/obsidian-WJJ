# 옵시디언 개선 제안

> 클로이가 분석한 WJJ의 옵시디언 Vault 개선 제안서

작성일: 2026-03-25

---

## 📊 현재 상태 분석

### ✅ 잘한 점
1. **명확한 목표 구조**
   - "도전과제, 업적"으로 인생 목표 체계화
   - "자립하기 (취업, 창업)"을 핵심 과제로 설정
   - 25개 세부 도전과제 존재

2. **체계적인 분류**
   - 공부 / 도전과제 / 메뉴얼 / 습관 / 문화생활로 명확히 구분
   - 세이브포인트 개념 도입 (진행 상황 저장)

3. **메뉴얼 시스템**
   - 노하우, 철학, 공부법 등 실용적인 정보 정리
   - 게임 관련 메뉴얼도 별도 관리

### ⚠️ 개선 필요한 점

1. **도전과제 진행 상황 부재**
   - 각 도전과제의 현재 진행 상황을 알 수 없음
   - 완료율, 우선순위 불명확

2. **습관 트래킹 시스템 미흡**
   - "매일 영단어" 등 습관이 있는데 체계적인 기록 없음
   - 연속 달성일, 성공률 추적 불가

3. **일정관리 시스템 부재**
   - 일정에 대한 설명만 있고 구체적인 관리 방법 없음
   - 데일리/위클리/먼슬리 계획 없음

4. **취업 준비 구체화 필요**
   - "취직.md"가 내용이 거의 없음
   - 이력서, 포트폴리오, 면접 준비 등 구체적인 계획 필요

## 🎯 개선 제안

### 1. 도전과제 진행 상황 추적 시스템

**제안:**
```markdown
# 도전과제: 취직

## 진행 상황
- [ ] 이력서 작성
- [ ] 포트폴리오 정리
- [ ] 기술 스택 결정
- [ ] 면접 준비
- [ ] 채용 공고 검색

## 우선순위
🔴 **긴급** (당장 해결 필요)
🟡 **중요** (3개월 내 목표)
🟢 **보통** (1년 내 목표)

## 관련 노트
- [[ai 공부]]
- [[개발자 포트폴리오]]
```

### 2. 습관 트래킹 시스템

**제안: 데일리 습관 트래커**
```markdown
# 📅 2026-03-25 습관 체크리스트

## 아침 루틴
- [ ] 기상 (07:00)
- [ ] 물 마시기
- [ ] 영단어 1시간
- [ ] 운동 30분

## 저녁 루틴
- [ ] 일기 작성
- [ ] 내일 계획
- [ ] 핸드폰 금지 (22:00 이후)

## 점수
- 🔥 연속 달성: 0일
- ✅ 오늘 완료: 0/7
```

### 3. 일정관리 시스템

**제안: 캘린더 + 프로젝트 시스템**
```markdown
# 일정 관리

## 📅 이번 주 (3월 4주차)
- [ ] 취업 박람회 참석 (3/28)
- [ ] 포트폴리오 완성 (3/31)
- [ ] 코딩 테스트 연습 (매일)

## 🎯 이번 달 목표
1. 이력서 초안 완성
2. GitHub 5개 프로젝트 업로드
3. 기술 면접 준비

## 📆 정기 일정
- 매일: 코딩 연습 1시간
- 매주: 포트폴리오 업데이트
- 매월: 이력서 검토
```

### 4. 취업 준비 로드맵

**제안:**
```markdown
# 취업 준비 로드맵

## 1단계: 기술 역량 (현재 ~ 2개월)
- [ ] 프로그래밍 언어 1개 마스터
- [ ] GitHub 포트폴리오 구축
- [ ] 코딩 테스트 연습

## 2단계: 서류 준비 (2개월 ~ 3개월)
- [ ] 이력서 작성
- [ ] 자기소개서 작성
- [ ] 포트폴리오 정리

## 3단계: 지원 (3개월 ~ 6개월)
- [ ] 채용 공고 모니터링
- [ ] 지원서 작성
- [ ] 면접 준비

## 4단계: 면접 (6개월 ~ )
- [ ] 모의 면접 연습
- [ ] 기술 면접 준비
- [ ] 최종 합격
```
