# 게임 일퀘

> 하루 루틴 완료 후 보상으로 게임 즐기기

**유형**: 🎁 보상 (동기부여)
**시간**: 30분-1시간
**목표**: 루틴 완료 후 즐기기

---

## 📋 보상 정보

### 언제
- 운동 & 샤워 후
- 하루 루틴 마무리 시점

### 얼마나
- 일퀘 완료할 때까지
- 보통 30분-1시간

### 왜
- 하루 노력에 대한 보상
- 동기부여
- 스트레스 해소

---

## 🎮 게임 목록

### 현재 플레이 중
- 젠존제
- 엔필
- 마작
- 메이플

### 일퀘 내용
- 젠존제: 데일리 퀘스트
- 엔필: 일일 미션
- 마작: 일일 대국
- 메이플: 일퀘 돌기

---

## 🎯 보상 시스템

### 기본 보상
- ✅ 루틴 완료 → 게임 일퀘 (자연스러운 보상)

### 연속 달성 보상
- 🔥 7일 연속: 게임 1시간 추가
- 🔥 14일 연속: 좋아하는 음식
- 🔥 30일 연속: 특별 보상

### 실패 시
- 루틴 미완료 → 게임 일퀘 못 함
- 8/10 이상 완료 → 게임 OK

---

## 💡 보상 규칙

### 엄격한 규칙
- ✅ 루틴 80% 이상 완료 → 게임 OK
- ❌ 루틴 80% 미만 → 게임 X

### 유연한 규칙
- 핵심 습관 3개 완료 → 게임 OK
- 특별한 상황 → 유연하게
