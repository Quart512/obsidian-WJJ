# 옵시디언 시작하기

> [!tip] 이 문서를 읽고 나면...
> 옵시디언이 무엇인지, 왜 사용하는지, 그리고 어떻게 시작하는지 이해하게 됩니다.

## 옵시디언이란 무엇인가

옵시디언(Obsidian)은 **로컬 마크다운 파일** 기반의 지식 관리 도구입니다. 2020년 출시 이후 전 세계 수백만 명의 연구자, 작가, 개발자, 학생들이 사용하고 있습니다.

### 다른 도구와의 차이점

| 특징 | 옵시디언 | Notion | Evernote |
|------|----------|--------|----------|
| 파일 저장 위치 | 로컬 (내 컴퓨터) | 클라우드 서버 | 클라우드 서버 |
| 파일 형식 | 마크다운 (.md) | 독자 포맷 | 독자 포맷 |
| 오프라인 사용 | 완전 지원 | 제한적 | 제한적 |
| 데이터 소유권 | 완전히 내 것 | 서비스 종속 | 서비스 종속 |
| 링크 기능 | 양방향 링크 | 단방향 | 없음 |
| 가격 | 무료 (기본) | 유료 플랜 | 유료 플랜 |

## 핵심 개념

### 볼트 (Vault)

볼트는 옵시디언이 관리하는 **폴더**입니다. 옵시디언을 열면 특정 폴더를 볼트로 지정하고, 그 안의 모든 마크다운 파일을 노트로 인식합니다.

```
내 볼트/
├── 00 가이드/
├── 01 프로젝트/
├── 02 영역/
├── 03 자료/
└── 홈.md
```

> [!important] 볼트 위치 선택
> 볼트 폴더는 iCloud, Dropbox, Google Drive 등 클라우드 동기화 폴더 안에 두면 자동으로 백업 및 동기화됩니다. 단, OneDrive는 파일 잠금 문제가 발생할 수 있어 주의가 필요합니다.

### 노트 (Note)

노트는 볼트 안의 **마크다운 파일(.md)** 하나하나입니다. 일반 텍스트 편집기로도 열 수 있으며, 옵시디언 없이도 내용을 확인할 수 있습니다.

### 내부 링크 (Internal Link)

노트와 노트를 연결하는 핵심 기능입니다. `[[노트 이름]]` 형식으로 작성하면 해당 노트로 이동하는 링크가 생성됩니다.

```markdown
오늘 읽은 [[아토믹 해빗]] 책에서 습관의 중요성을 배웠다.
```

### 백링크 (Backlink)

어떤 노트를 다른 노트에서 링크했을 때, **링크된 노트에서 역방향으로 누가 나를 링크했는지** 확인할 수 있습니다. 이것이 옵시디언의 가장 강력한 기능 중 하나입니다.

### 그래프 뷰 (Graph View)

모든 노트와 링크를 **시각적 네트워크**로 보여주는 기능입니다. 노트들 사이의 연결 관계를 한눈에 파악할 수 있습니다.

- `Ctrl/Cmd + G` : 전체 그래프 열기
- 특정 노트에서 **로컬 그래프**를 열면 해당 노트와 연결된 노트들만 볼 수 있습니다

## 기본 사용법

### 첫 번째 노트 만들기

1. `Ctrl/Cmd + N` 을 눌러 새 노트를 만듭니다
2. 제목을 입력합니다 (파일명이 됩니다)
3. 내용을 마크다운으로 작성합니다

### 노트 간 이동

- `[[` 를 입력하면 노트 검색 창이 열립니다
- 노트 이름을 타이핑하면 자동완성됩니다
- `]]` 로 닫으면 링크가 완성됩니다

### 주요 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl/Cmd + N` | 새 노트 만들기 |
| `Ctrl/Cmd + O` | 노트 빠른 열기 |
| `Ctrl/Cmd + P` | 커맨드 팔레트 |
| `Ctrl/Cmd + G` | 그래프 뷰 열기 |
| `Ctrl/Cmd + E` | 편집/미리보기 전환 |
| `Ctrl/Cmd + ,` | 설정 열기 |
| `Ctrl/Cmd + Shift + F` | 전체 검색 |
| `Alt + Enter` | 링크된 노트 새 창에서 열기 |

### 사이드바 활용

- **왼쪽 사이드바**: 파일 탐색기, 검색, 북마크
- **오른쪽 사이드바**: 백링크, 아웃고잉 링크, 태그 목록

## 옵시디언의 철학

### 1. 로컬 파일 우선

모든 노트는 내 컴퓨터의 폴더에 `.md` 파일로 저장됩니다. 옵시디언 서비스가 종료되어도, 인터넷이 없어도 내 데이터는 안전합니다.

> [!quote] 옵시디언의 약속
> "우리는 당신의 데이터를 인질로 잡지 않습니다. 언제든지 다른 도구로 옮길 수 있습니다."

### 2. 마크다운의 힘

마크다운은 1970년대 이후 사용되어 온 단순한 텍스트 형식입니다. 앞으로도 수십 년간 읽힐 수 있는 **미래 보장(future-proof)** 형식입니다.

```markdown
# 이것이 마크다운입니다
**굵게**, *기울임*, [[내부링크]], #태그
```

### 3. 연결의 힘

정보는 고립되면 쓸모가 없습니다. 노트들이 서로 연결될 때, 새로운 통찰이 생겨납니다. 이것이 옵시디언이 단순한 메모 앱과 다른 이유입니다.

> [!tip] 제텔카스텐의 핵심
> 노트를 폴더로 분류하는 것보다 **링크로 연결하는 것**이 훨씬 강력합니다. 하나의 아이디어는 여러 맥락에서 동시에 존재할 수 있습니다.

### 4. 오래 지속되는 노트

노트는 한 번 쓰고 버리는 것이 아닙니다. 시간이 지나면서 내용을 추가하고, 다른 노트와 연결하고, 정제해 나가는 **살아있는 문서**입니다.
