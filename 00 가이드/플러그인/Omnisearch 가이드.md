# Omnisearch 플러그인 가이드

> 전체 텍스트 검색 강화

**카테고리**: 노트 탐색 & 정리
**난이도**: 낮음
**추천도**: ⭐⭐⭐⭐

---

## 📋 개요

### **Omnisearch란?**
옵시디언의 기본 검색을 강화하여 PDF, 헤딩, 파일명, 모든 노트 내용을 더 정확하게 검색하는 플러그인

### **주요 기능**
- ✅ 전체 텍스트 검색
- ✅ PDF 검색
- ✅ 헤딩 검색
- ✅ 키워드 하이라이트
- ✅ 미리보기
- ✅ 검색 결과 순위 매기기

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Omnisearch"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Omnisearch
- Index PDF files: ON
- Cache: ON
- Highlight results: ON
```

---

## 🎨 기본 사용법

### **1. 검색 실행**
```
Ctrl/Cmd + O → 검색창 열기
또는
Command Palette → Omnisearch: Search
```

### **2. 검색 결과**
```
- 파일명
- 헤딩
- 내용 미리보기
- 키워드 하이라이트
```

---

## ⚡ 고급 기능

### **1. PDF 검색**
```
Settings → Omnisearch
- Index PDF files: ON
- Respect Excluded Files: ON
```

### **2. 캐시 관리**
```
Settings → Omnisearch → Advanced
- Refresh index: 버튼 클릭
- Clear cache: 버튼 클릭
```

### **3. 제외 설정**
```
Settings → Omnisearch → Excluded folders
- Templates
- Archive
```
