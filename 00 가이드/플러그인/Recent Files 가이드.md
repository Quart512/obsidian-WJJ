# Recent Files 플러그인 가이드

> 최근 열어본 파일 빠르게 접근

**카테고리**: 노트 탐색 & 정리
**난이도**: 낮음
**추천도**: ⭐⭐⭐

---

## 📋 개요

### **Recent Files란?**
가장 최근에 열었던 파일 목록을 패널에 표시하여 빠르게 이동할 수 있는 플러그인

### **주요 기능**
- ✅ 최근 파일 목록
- ✅ 클릭으로 바로 열기
- ✅ 파일 간 빠른 전환
- ✅ 목록 길이 설정

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Recent Files"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Recent Files
- List length: 20 (기본값)
- Show file extensions: ON/OFF
```

---

## 🎨 기본 사용법

### **1. 패널 열기**
```
리본 아이콘 클릭 → Recent Files 패널
```

### **2. 파일 열기**
```
파일명 클릭 → 바로 열기
```

---

## 🎯 WJJ 활용 예시

### **1. 자주 여는 파일 관리**
```
- 데일리 노트
- 습관 트래커
- 로드맵
- 게임 개발 노트
```

### **2. 워크플로우 최적화**
```
1. Recent Files에서 파일 열기

```
