# HoverNotes 플러그인 가이드

> 비디오 학습을 위한 타임스탬프 노트

**카테고리**: 학습 & 시각화
**난이도**: 중간
**추천도**: ⭐⭐⭐

---

## 📋 개요

### **HoverNotes란?**
YouTube, Udemy 등 비디오 콘텐츠에서 타임스탬프가 찍힌 노트와 스크린샷을 마크다운으로 직접 캡처하는 플러그인

### **주요 기능**
- ✅ 타임스탬프 노트
- ✅ 스크린샷 캡처
- ✅ YouTube 연동
- ✅ Udemy 연동
- ✅ 마크다운 변환

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "HoverNotes"
Install → Enable
```

### **2. 기본 설정**
```
Settings → HoverNotes
- Default folder: "Video Notes"
- Screenshot format: PNG
```

---

## 🎨 기본 사용법

### **1. 비디오 열기**
```
HoverNotes → Open Video
URL 붙여넣기
```

### **2. 타임스탬프 추가**
```
재생 중 단축키 → 타임스탬프 생성
```

### **3. 스크린샷 캡처**
```
단축키 → 현재 프레임 캡처
```
