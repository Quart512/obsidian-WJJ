## 📋 개요

### **Commander란?**
옵시디언 UI의 다양한 섹션에 커스텀 버튼을 추가하여 자주 사용하는 기능에 빠르게 접근할 수 있는 플러그인

### **주요 기능**
- ✅ 커스텀 버튼 추가
- ✅ 명령어 연결
- ✅ 다양한 위치 지원
- ✅ 아이콘 설정

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Commander"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Commander
- Add new command
```

---

## 🎨 버튼 만들기

### **1. 명령어 선택**
```
Settings → Commander → Add Command
- 명령어 선택
- 아이콘 선택
- 위치 선택
```

### **2. 위치 설정**
```
- Ribbon: 리본 메뉴
- Titlebar: 타이틀바
- Sidebar: 사이드바
- Statusbar: 상태바
```

---

## 🎯 WJJ 활용 예시

### **1. 자주 쓰는 명령어**
```
- Templater: 새 데일리 노트
- QuickAdd: 습관 로그
```
