# Custom Frames 플러그인 가이드

> 웹페이지를 옵시디언 내에 임베드

**카테고리**: 유틸리티 & UI 개선
**난이도**: 낮음
**추천도**: ⭐⭐⭐

---

## 📋 개요

### **Custom Frames란?**
원하는 웹페이지를 옵시디언 내에 임베드하여 사용할 수 있는 플러그인

### **주요 기능**
- ✅ 웹페이지 임베드
- ✅ Notion 임베드
- ✅ Google Calendar 임베드
- ✅ 커스텀 프레임 생성

---

## 🚀 설치 & 설정

### **1. 설치**
```
Settings → Community plugins → Browse
검색: "Custom Frames"
Install → Enable
```

### **2. 기본 설정**
```
Settings → Custom Frames
- Add new frame
```

---

## 🎨 프레임 만들기

### **1. Google Calendar**
```
Name: Google Calendar
URL: https://calendar.google.com
```

### **2. Notion**
```
Name: Notion
URL: https://notion.so
```

### **3. 개발 문서**
```
Name: React Docs
URL: https://react.dev
```
