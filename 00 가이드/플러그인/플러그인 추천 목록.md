## 🛠️ 설치 방법

### **1. 커뮤니티 플러그인 활성화**
```
Settings → Community plugins → Restricted mode 끄기
```

### **2. 플러그인 검색 & 설치**
```
Browse → 검색 → Install → Enable
```

### **3. 설정 파일 위치**
```
.obsidian/
├── plugins/
│   ├── dataview/
│   ├── tasks/
│   └── templater-obsidian/
```

## 💡 활용 팁

### **Templater + Dataview 조합**
```markdown
# 오늘의 습관 진행 상황

```dataview
TABLE
  choice(영단어, "✅", "❌") as "영단어",
  choice(운동, "✅", "❌") as "운동"
FROM "일정/데일리"
WHERE file.name = "<% tp.date.now('YYYY-MM-DD') %>"
```
```

### **QuickAdd 매크로**
```bash
# 습관 로그 추가 매크로
1. 새 노트 생성
2. 템플릿 적용
3. 날짜 자동 입력
4. 체크리스트 추가
```
