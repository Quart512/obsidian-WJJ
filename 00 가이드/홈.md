# 내 옵시디언 볼트

## 빠른 시작

옵시디언을 처음 사용한다면 이 순서로 읽어보세요:

1. [[00 가이드/옵시디언 시작하기|옵시디언 시작하기]] — 옵시디언이 무엇인지, 핵심 개념 이해
2. [[00 가이드/마크다운 작성법|마크다운 작성법]] — 노트 작성에 필요한 문법 익히기
3. [[00 가이드/추천 플러그인|추천 플러그인]] — 옵시디언을 강력하게 만드는 플러그인 설치
4. [[00 가이드/메모 관리 시스템|메모 관리 시스템]] — 체계적인 노트 관리 방법론 선택
5. [[00 가이드/학습에 옵시디언 활용하기|학습에 옵시디언 활용하기]] — 공부와 연구에 활용하기
6. [[00 가이드/템플릿 모음|템플릿 모음]] — 바로 쓸 수 있는 노트 템플릿 복사

## 볼트 구조

```
볼트/
├── 00 가이드/        ← 옵시디언 사용법 가이드
├── 01 프로젝트/      ← 진행 중인 프로젝트 (만들어서 사용하세요)
├── 02 영역/          ← 지속 관리하는 책임 영역
├── 03 자료/          ← 참고 자료, 독서 노트
├── 04 보관/          ← 완료된 프로젝트, 과거 자료
├── 99 템플릿/        ← 노트 템플릿 (Templater 설정 필요)
├── Daily/            ← 일일 노트
└── 홈.md            ← 지금 이 파일 (메인 허브)
```

> [!tip] 폴더 직접 만들기
> 위 폴더들은 직접 만들어야 합니다. 왼쪽 파일 탐색기에서 우클릭 → 새 폴더를 선택하세요.

````dataview

WHERE file.name != "홈"
SORT file.mtime DESC
````
