```
✅ 유선형 디자인
```

### **선호 기체:**
```

✅ 엑시아

✅ F91
```

### **비선호 디자인:**
```

```

```
✅ 3차원적인 덩어리감이 확실한 제트팩
✅ 추진기나 무기로서의 기계적 기능과 목적이 명확한 형태
✅ 입체적이고 실용적인 디자인
```

### **선호 기체:**

### **비선호 디자인:**
```
❌ 에일 스트라이크, 윙 건담처럼 얇고 평면적으로 넓게 펼쳐지는 장식적인 천사 날개
❌ 장식적인 형태
```

## 3. 슬림한 본체와 거대 무장의 폭발적인 시너지 (무장) ⭐⭐⭐⭐⭐

### **선호 디자인:**
```
✅ 날렵한 본체가 자신의 스케일을 상회하는 거대한 포나 대함도를 다루는 언밸런스한 조합
✅ 날카로움 속에서 압도적인 화력을 뿜어내는 '응축된 강함'
```

### **선호 기체:**
```
✅ 엑시아 리페어 3의 롱 라이플
✅ 슬림한 본체 + 거대 무장 조합
```

### **비선호 디자인:**
```
❌ 뚱뚱하고 둔해 보이는 기체
```

## 4. 기계적 생동감을 주는 외부 노출형 프레임 (디테일) ⭐⭐⭐⭐⭐

### **선호 디자인:**
```
✅ 장갑 밖으로 고스란히 드러난 날 것 그대로의 메카닉 디테일
✅ 기계적인 리얼리티
✅ 장갑 틈새로 보이는 차가운 금속성의 서스펜션과 뼈대의 질감
```

### **선호 기체:**
```
✅ 건담 발바토스
✅ 건담 비다르
✅ 관절부 실린더(피스톤)나 동력선(파이프)이 장갑 밖으로 노출된 기체
```

### **특징:**
```
✅ 내부 프레임 구조를 상당 부분 공유
✅ 관절이 움직일 때마다 파이프와 피스톤이 연동
✅ 정밀하고 공학적인 취향을 완벽하게 충족
```
