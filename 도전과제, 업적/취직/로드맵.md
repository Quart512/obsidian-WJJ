## 📋 4단계 로드맵

### 1️⃣ 1단계: 기술 역량 강화 (현재 ~ 4월 말)

#### 프로그래밍 기초
- [ ] **프로그래밍 언어 선택** (우선순위: 높음)
  - Python, JavaScript, TypeScript 중 선택
  - 선택 후 깊이 있게 학습
  
- [ ] **코딩 테스트 준비**
  - 백준/프로그래머스 매일 1문제
  - 알고리즘 기초 (정렬, 탐색, DP)
  - 자료구조 (배열, 리스트, 트리, 그래프)

#### 개발 프로젝트
- [ ] **GitHub 포트폴리오 구축**
  - 개인 프로젝트 2개 이상
  - README 작성법 학습
  - 코드 퀄리티 높이기
  
- [ ] **기술 스택 결정**
  - 프론트엔드 vs 백엔드 vs 풀스택
  - 관심 분야: AI, 웹개발, 게임개발

#### 학습 리소스
- [ ] 온라인 강의 수강 (인프런, Udemy)
- [ ] 기술 블로그 작성 시작
- [ ] 개발 커뮤니티 참여

### 2️⃣ 2단계: 서류 준비 (5월 초 ~ 5월 말)

#### 이력서
- [ ] **이력서 초안 작성**
  - 개발자용 이력서 템플릿 찾기
  - 프로젝트 경험 정리
  - 기술 스택 명시
  
- [ ] **이력서 검토 & 수정**
  - 멘토/커뮤니티 피드백
  - 디자인 개선

#### 포트폴리오
- [ ] **GitHub 정리**
  - 프로젝트별 README 완성
  - 코드 주석 추가
  - 데모 영상/스크린샷 추가
  
- [ ] **개인 포트폴리오 사이트** (선택)
  - GitHub Pages or Netlify
  - 프로젝트 소개 페이지

#### 자기소개서
- [ ] **자기소개서 작성**
  - 개발자로서의 동기
  - 성장 가능성 어필
  - 프로젝트 경험 서술

- [ ] **기업 리스트 작성**
  - 지원할 기업 20개 선정
  - 우선순위 정리
  - 기업별 맞춤 전략

#### 지원 준비
- [ ] **지원서 작성**
  - 기업별 맞춤 이력서
  - 커버레터 작성법 학습
  
- [ ] **지원 타이밍 계획**
  - 지원 순서 정하기
  - 동시 지원 vs 순차 지원

### 4️⃣ 4단계: 면접 준비 & 합격 (6월 중 ~ )

  - 화이트보드 코딩 연습

- [ ] **면접 스케줄 관리**

- [ ] **최종 합격**
